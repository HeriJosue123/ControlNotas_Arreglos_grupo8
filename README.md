# Tarea Asíncrona – Registro de Notas con Arreglos

## Descripción del proyecto
Este proyecto corresponde a una aplicación de consola desarrollada en C#, cuyo objetivo es registrar las notas de estudiantes y calcular su promedio individual, cumpliendo con el uso exclusivo de arreglos, según lo solicitado en la actividad.

## Funcionamiento del programa
El programa trabaja con un total de 5 estudiantes y registra 3 notas por cada uno.  
Los datos son solicitados por teclado y almacenados en arreglos unidimensionales y bidimensionales.

## Validaciones implementadas
Durante la ejecución del programa se validan los siguientes aspectos:
- El nombre del estudiante no puede estar vacío.
- El nombre debe contener al menos 3 caracteres.
- El nombre no puede incluir números.
- Las notas solo aceptan números enteros.
- Las notas deben estar en un rango de 0 a 10.

Estas validaciones garantizan el correcto ingreso de datos y evitan errores durante el cálculo de los promedios.

## Estructura utilizada
- Arreglo unidimensional para almacenar los nombres de los estudiantes.
- Arreglo bidimensional para almacenar las notas.
- Arreglo unidimensional para almacenar los promedios.
- Uso de ciclos `for` para el ingreso de datos, cálculo de promedios y visualización de resultados.

## Resultados
Al finalizar la ejecución, el programa muestra en consola:
- Las notas ingresadas para cada estudiante.
- El promedio final correspondiente a cada estudiante.

## Tecnologías utilizadas
- Lenguaje de programación: C#
- Tipo de aplicación: Consola
- Framework: .NET Framework 4.8
